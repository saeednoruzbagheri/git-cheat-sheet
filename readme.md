# git cheat sheet

```bash

$ git init #initialize


$ git status # status of your repo


$ git add [file] # add spetion file to stage mode
or
$ git add -A # add all file to stage mode



$ git commit -m "{massage}" # commit changes


$ git log # see all commits

$ git diff HEAD # see diff of chenges file

$ git reset [file] #unstage my file

$ git checkout -- [file] # dont chnge my file and set it as last commit


```
